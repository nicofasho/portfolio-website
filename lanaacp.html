Louisiana NAACP

Project Overview

This project was a website for the Louisiana NAACP. I solo designed all visual elements: color, typography, layout, and images. I also coded it in HTML, CSS, and Javascript. I added additional features by connecting them to services like Eventbrite so they could accept registrations for their annual conference.
